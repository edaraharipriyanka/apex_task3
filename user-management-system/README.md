# 🚀 User Management System

A secure and responsive **User Management System** built using **PHP, MySQL, Bootstrap, HTML, CSS, JavaScript, and XAMPP** as part of **Task 3 – Backend Development & Database Integration** during my **Web Development Internship at Apex Planet Software Pvt. Ltd.**

---

## 📖 Overview

This project demonstrates backend development concepts including **authentication, CRUD operations, database integration, profile management, and security**. It features role-based access for **Admin** and **User**, ensuring secure and efficient user management.

---

## ✨ Features

### 👤 Authentication

* User Registration & Login
* Role-Based Access (Admin/User)
* Secure Logout
* Password Hashing & Verification
* Session Management

### 📂 User Management

* Create, Read, Update & Delete (CRUD)
* Search Users
* User Statistics Dashboard
* Profile Image Upload
* Edit Profile
* Change Password

### 🔒 Security

* Prepared Statements (`mysqli_prepare`)
* SQL Injection Prevention
* Server-side Validation
* Duplicate Email Validation
* Role-Based Authorization

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/edaraharipriyanka/apex_task3.git
```

2. Move the project to:

```text
xampp/htdocs/
```

3. Start **Apache** and **MySQL** in XAMPP.

4. Create a database named:

```text
user_management
```

5. Import the SQL file into **phpMyAdmin**.

6. Configure the database in `config/database.php`.

7. Run the project:

```text
http://localhost/user-management-system/
```
